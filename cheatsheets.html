Cheatsheets

It's been said that experience isn't about having all the formulas memorized and available for instant recollection at any time. It's more about knowing where to look for the syntax and formulas you need. Having the right cheatsheets and resources on hand and knowing when to consult them is a practice that will stay with you for the entirety of your career!

To that end, I've put together a compilation of cheatsheets here that I've found useful:

  • Data Science Cheatsheets on GitHub. This fantastic cheatsheet list compiled by GitHub user abhat222 covers virtually every aspect of Data Science. This cheatsheet compilation is every bit the gold mine it sounds like and then some!
  • RStudio Cheatsheets. Cheatsheets for the R program provided by the RStudio website! It's great to have resources like these directly from the program's creators. You'll find cheatsheets for most libraries and packages of interest here.
  • 101 Machine Learning Algorithms. Useful cheat-sheets and explanations of various Machine Learning algorithms (with easy to follow examples!)
